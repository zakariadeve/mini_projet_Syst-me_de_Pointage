## ✅ اختبار 2: التحقق من API

افتح هذا الرابط في المتصفح (قبل إغلاق التطبيق):

```
http://localhost:5000/api/statut
```

### النتيجة المتوقعة:
```json
{
  "status": "active",
  "database_exists": true,
  "total_etudiants": 5,
  "database_size_bytes": 16384
}
```

**تأكد من**:
- [] `"status": "active"`
- [] `"database_exists": true`
- [] `"total_etudiants": 5` (أو العدد الذي أضفته)
- [] `"database_size_bytes"` > 8192

احفظ هذه الرسالة (احفظ الصفحة `Ctrl+S`) كمرجع.

## ✅ اختبار 6: فحص الملفات

افتح PowerShell في مجلد المشروع:

```powershell
# تحقق من وجود ملف البيانات
Get-ChildItem | Where-Object {$_.Name -eq "etudiants.db"}

# العرض المتوقع:
# Mode: -a----
# Length: 16384 (أو أكثر)
# Name: etudiants.db
```

### النتيجة المتوقعة:
- [] الملف موجود
- [] الحجم > 8192 بايت
- [] تاريخ التعديل حديث (اليوم)

## ⚠️ استكشاف الأخطاء الشائعة

### الخطأ 1: تظهر "❌ Aucun étudiant" بعد إعادة التشغيل

**التشخيص**:
```bash
# افتح PowerShell
cd d:\TSDI2éme\python\mini_projet

# تحقق من الملف
dir etudiants.db

# إذا لم يظهر، المشكلة في إنشاء الملف
```

**الحل**:
1. احذف مجلد `__pycache__`:
   ```bash
   rmdir /s __pycache__
   ```
2. أعد تشغيل التطبيق
3. أضف طالب واحد
4. أغلق وأعد التشغيل

### الخطأ 2: رسالة "Database is locked"

**الحل**:
1. أغلق المتصفح تماماً
2. https://localhost:5000 في نافذة جديدة
3. إذا استمر، احذف `etudiants.db` وأعد البدء

### الخطأ 3: الـ API `/api/statut` يعرض عدد مختلف

**الحل**:
1. انتظر 2 ثانية (قد يكون مؤقتاً)
2. أعد تحميل الصفحة (`F5`)
3. حاول في نافذة جديدة (إذا كانت هناك مشاكل cache)
