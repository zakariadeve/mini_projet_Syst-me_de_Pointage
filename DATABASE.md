## 🚀 روات الواجهة (Web Routes)

### 1. **الصفحات الرئيسية**
```
/                      → الصفحة الرئيسية (قائمة الحضور)
/ajouter              → صفحة إضافة طالب جديد
```

```
/pointage/<nom>       → إضافة طالب مباشرة من الـ URL

```

```

```

## 📡 API Routes (JSON)

هذه الروات توفر بيانات بصيغة JSON للتطبيقات الأخرى:

### 1. **الحصول على جميع الطلاب**
```
GET /api/etudiants
```

**الاستجابة**:
```json
[
  {
    "id": 1,
    "nom": "Ahmed Ali",
    "heure": "14:35:42",
    "date": "2026-03-05"
  },
  {
    "id": 2,
    "nom": "Fatima Bennani",
    "heure": "14:36:15",
    "date": "2026-03-05"
  }
]
```

### 2. **تصدير البيانات الكاملة**
```
GET /api/export
```

**الاستجابة**:
```json
{
  "total": 2,
  "export_date": "2026-03-05 14:37:00",
  "etudiants": [...]
}
```

### 3. **حالة قاعدة البيانات**
```
GET /api/statut
```

**الاستجابة**:
```json
{
  "status": "active",
  "database_file": "D:\\TSDI2éme\\python\\mini_projet\\etudiants.db",
  "database_exists": true,
  "database_size_bytes": 8192,
  "total_etudiants": 2,
  "last_update": "2026-03-05 14:37:00"
}
```

### 4. **حذف طالب معين**
```
POST /api/supprimer/<id>
```

**مثال**: `POST /api/supprimer/1`

**الاستجابة**:
```json
{
  "success": true,
  "message": "Étudiant Ahmed Ali supprimé"
}
```

## 🔍 كيفية التتبع والتحقق

### اختبار حفظ البيانات

1. **إضافة طلاب**
   - اذهب إلى http://localhost:5000
   - أضف عدة طلاب
   - تأكد من ظهورهم في الجدول

2. **إعادة التشغيل**
   - أوقف التطبيق (Ctrl+C)
   - انتظر 2 ثانية
   - أعد تشغيل التطبيق
   - تأكد من وجود البيانات السابقة

3. **التحقق من الـ API**
   - افتح http://localhost:5000/api/statut في المتصفح
   - تأكد من أن `database_exists` = `true`
   - تأكد من أن `total_etudiants` محدث

4. **تصدير البيانات**
   - افتح http://localhost:5000/api/export
   - انسخ الـ JSON كنسخة احتياطية

## 🛠️ استكشاف الأخطاء

### مشكلة: البيانات لا تظهر بعد الإعادة

**الحل**:
```bash
# تحقق من وجود ملف البيانات
dir etudiants.db

# إذا لم يكن موجوداً، احذف ملف __pycache__
rmdir /s __pycache__

# أعد تشغيل التطبيق
python app.py
```

### مشكلة: خطأ في Database

**الحل**:
1. افتح http://localhost:5000/api/statut
2. تأكد من `"status": "active"`
3. إذا كانت هناك مشكلة، احذف `etudiants.db` وأعد التشغيل

### مشكلة: قاعدة البيانات تالفة

**الحل**:
```bash
# حذف البيانات القديمة
del etudiants.db

# إعادة التشغيل (سينشئ قاعدة جديدة)
python app.py
```
